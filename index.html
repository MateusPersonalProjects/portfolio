About Me

Hello, my name is Mateus, I'm 26 years old and a Computer Science student at the Federal University of Paraná (UFPR)! My first contact with development was at the age of 14 when I joined a technical school, there I was introduced to various programming languages such as C++, Java, PHP, JavaScript, and was able to work on various web development projects. Currently, I have been focusing my studies on Data Science with Python (Pandas, Matplotlib, Seaborn, scikit-learn, PyTorch, and TensorFlow) and Back-End Development using Python and JavaScript with Node.js. Shopee Finder

Shopee Finder was created with the intention of helping collectors find collectibles more easily on Shopee. Having to look at the website multiple times for new products can be tiring when done for many days, weeks, and months. In order to solve this problem, I created Shopee Finder, an assistant for your searches on Shopee! Shopee Finder was built entirely in Python and uses Selenium, Beautiful Soup to perform web scraping of the site and inform you about new products via email. Additionally, it will generate a CSV file with all the items found in the search that can be accessed directly from the project folder. Feel free to use the project! If you have any questions or comments, please feel free to contact me.

Flight Finder

Flight Finder was created with the goal of helping you find cheaper flights. The project was built using various APIs such as Kiwi (to find flights), Twilio to notify the user with SMS when a new flight is found, and Sheety to store the locations the user wants to go to. Feel free to use the project!
